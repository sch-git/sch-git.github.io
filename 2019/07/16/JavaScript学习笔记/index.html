变量

一个变量，就是一个用于存放数值的容器。这个数值可能是一个用于累加计算的数字，或者是一个句子中的字符串。变量的独特之处在于它存放的数值是可以改变的。
javascript中用var声明变量
var x;
var str;
初始化变量
x=1;
str=’Hello World!’;//单双引号几乎没有区别
str=”Hello World!”;
